# AI-Powered Social Media Automation using Make.com

This project automates **personalized social media content creation and posting** using **Make.com**, **Google Gemini 2.5 Flash**, and **Google Sheets**.

The workflow acts as a **personalized content engine** that reads input links and data from Google Sheets, generates customized posts using AI, and automatically publishes them to multiple social media platforms on a scheduled basis.

## ğŸ” Workflow Description

### Step 1: Content Input
- User adds:
  - URL / achievement / topic
  - Platform preference
  - Posting schedule (date/time or frequency)
- Data is stored in **Google Sheets**

### Step 2: AI Personalization
- Google Gemini 2.5 Flash generates:
  - Platform-specific tone
  - Personalized captions
  - Optimized formatting per social media channel

### Step 3: Routing & Platform Logic
- Make.com **Router** directs content based on:
  - Selected platform(s)
  - Posting conditions

### Step 4: Automated Posting
- Posts are published automatically to:
  - LinkedIn
  - Facebook
  - Telegram
- Supports:
  - Scheduled posts
  - One-time posts
  - Recurring daily/interval-based posts
